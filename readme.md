# Node changer

Requires fnm pre-installation [fnm](https://github.com/Schniz/fnm)

Switch node version based on Fast Node Manager (fnm). it grabs commands:
fnm install
fnm use
fnm default

## Installation

```
npm i -g node-changer
```

## Usage:

```
node-changer
```
